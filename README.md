# Bistrot Arlésien - Site Web Officiel

Site web one-page pour le restaurant Bistrot Arlésien situé à Arles (Place du Forum).

## 🚀 Déploiement sur GitHub Pages

Ce site est prêt pour un déploiement immédiat sur GitHub Pages.

### Instructions de déploiement :

1. **Créer un nouveau repository GitHub**
   - Nom suggéré : `bistrot-arlesien`
   - Public ou Privé selon votre préférence

2. **Copier le fichier index.html**
   - Le fichier `index.html` à la racine de ce projet contient tout le site
   - C'est le SEUL fichier nécessaire pour le déploiement

3. **Push vers GitHub**
   ```bash
   git init
   git add index.html
   git commit -m "Site Bistrot Arlésien"
   git branch -M main
   git remote add origin https://github.com/VOTRE_USERNAME/bistrot-arlesien.git
   git push -u origin main
   ```

4. **Activer GitHub Pages**
   - Aller dans Settings > Pages
   - Source : Deploy from a branch
   - Branch : main / (root)
   - Save

5. **Votre site sera disponible à :**
   `https://VOTRE_USERNAME.github.io/bistrot-arlesien/`
